# React For Ruby on Rails Developers

## Tech
- React
- Ruby 2.6.3
- Rails 6.0.2
- Bootstrap
- PostgreSQL

## Configuration
- Clone this repo
- Setup your database on `config/database.yml`
- Run `bundle install`
- Run `yarn install`
- Run `rails db:create` `db:migrate` and `rails db:seed` or `rails db:setup`
- Start server `rails server` or `rails s`
- Open localhost:3000
