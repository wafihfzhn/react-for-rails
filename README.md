# React For Ruby on Rails Developers

## Tech
- React
- Ruby 2.6.3
- Rails 6.0.2
- Bootstrap
- PostgreSQL

## Configuration
- Clone this repo
- Setup with your database on 'congif/database.yml'
- Run 'bundle install'
- Run 'yarn install'
- Run 'rails db:create' and 'rails db:seed' or 'rails db:setup'
- Start server 'rails server' or 'rails s'
- Open localhost:3000
